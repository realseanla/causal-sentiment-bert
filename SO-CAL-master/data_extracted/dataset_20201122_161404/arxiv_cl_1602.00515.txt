People are producing more written material then anytime in the history.
The increase is so high that professionals from the various fields are no more able to cope with this amount of publications.
Text mining tools can offer tools to help them and one of the tools that can aid information retrieval and information extraction is semantic text annotation.
In this report we present Marvin, a text annotator written in Java, which can be used as a command line tool and as a Java library.
Marvin is able to annotate text using multiple sources, including WordNet, MetaMap, DBPedia and thesauri represented as SKOS.
