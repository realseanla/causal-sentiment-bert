IDP is a knowledge base system based on first order logic.
It is finding its way to a larger public but is still facing practical challenges.
Adoption of new languages requires a newcomer-friendly way for users to interact with it.
Both an online presence to try to convince potential users to download the system and offline availability to develop larger applications are essential.
We developed an IDE which can serve both purposes through the use of web technology.
It enables us to provide the user with a modern IDE with relatively little effort.
